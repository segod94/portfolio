What is Full Stack Development?

Full stack means working on the whole app — what users see, the server logic behind it, and the database that stores the data.

Frontend

Everything the user sees and interacts with. Built with HTML for structure, CSS for styling, and JavaScript for behaviour.

HTML CSS JavaScript

Backend

The server side. Handles business logic, authentication, and APIs. Receives requests from the frontend and responds with data.

Node.js REST APIs Auth

Database

Where data lives. SQL for structured tables, NoSQL for flexible documents. The backend reads and writes here.

SQL NoSQL Storage
